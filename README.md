# Build from Scratch Journey

A public learning repo for building systems from scratch, starting with smaller fundamentals and progressing towards deeper systems such as servers, databases, compilers, kernels, and eventually an operating system.

The purpose of this repo is to understand how software works beneath the abstraction layers we normally rely on. Rather than treating tools and platforms as black boxes, I want to build simplified versions myself and learn by doing.

## Why this repo exists

I want a structured path into lower level systems thinking.

This repo is designed to help me:

- understand core software abstractions more deeply
- learn how protocols, runtimes, storage, and operating systems work
- improve my systems intuition
- build projects that show depth, curiosity, and engineering discipline

## Learning approach

The journey starts with smaller, realistic components and gradually moves towards more complex systems. The idea is to build confidence and understanding in layers, not jump straight to the hardest projects too early.

## What this repo covers

### 01 CLI Tools
Small command line tools to build comfort with inputs, outputs, files, and program structure.

### 02 Shell and Terminal Toys
Simple shell style experiments to understand processes, commands, pipes, and environment behaviour.

### 03 Parser and Interpreter Basics
Tokenising, parsing, simple grammars, and tiny interpreters to build language and compiler intuition.

### 04 HTTP from Scratch
Sockets, request parsing, response formatting, headers, status codes, and protocol basics.

### 05 Web Server from Scratch
Routing, middleware, static files, simple authentication, and server side request handling.

### 06 Database from Scratch
A small storage engine or key value style system to explore persistence, indexing, and data layout.

### 07 Cache from Scratch
In memory caching, TTL, eviction ideas, and the trade off between speed and freshness.

### 08 Message Queue from Scratch
Producer consumer patterns, buffering, retries, ordering, and durable messaging concepts.

### 09 Search Engine from Scratch
Tokenisation, inverted indexes, ranking basics, and retrieval ideas.

### 10 Container Runtime Basics
Core ideas behind process isolation, namespaces, cgroups, and container behaviour.

### 11 Compiler Basics
Lexing, parsing, syntax trees, and how source code becomes something executable.

### 12 File System Basics
Blocks, metadata, directories, and the fundamentals of file storage organisation.

### 13 Memory Management Basics
Stack vs heap, allocation, paging concepts, and memory related design trade offs.

### 14 Kernel Basics
Scheduling, interrupts, system calls, and the core responsibilities of an operating system kernel.

### 15 OS from Scratch
A long term stretch goal that builds on everything before it.

### notes
Reference notes, diagrams, links, and short explanations collected along the way.
